You should have X amount of districts, amount of people for those districts should be equal So we have a grid of units. Every unit A-people voted for your candidate and B-people voted for another candidate. Your mission is to unite all of those units in X amount of districts so your candidate would have more districts voted for him.

Input: Two arguments. Amount of people for each district (as integer) , grid of all electoral zone (as list of list (each unit has [A-peoples, B-peoples]))

Output: united districts (as List of string). The same letters of string represents the same district. If your candidate can not win more, return empty list([]).

Example: 

unfair_districts(5, [[[2, 1], [1, 1], [1, 2]],
                     [[2, 1], [1, 1], [0, 2]]]) == ['AAC',
                                                    'BBC]


unfair_districts(9, [[[0, 3], [3, 3], [1, 1]],
                     [[1, 2], [1, 0], [1, 1]],
                     [[0, 3], [2, 1], [2, 2]]]) == ['ABB',
                                                    'ABC',
                                                    'ACC']

Precondition:

  • 2 ≤ grid_rows, grid_columns ≤ 6
  • 4 ≤ grid_rows * grid_columns ≤ 25
  • Total_population % amuout_of_people == 0
    • Total_population == sum(sum(unit) for unit in chain(*grid))
  • 0 ≤ each_votes ≤ 9
